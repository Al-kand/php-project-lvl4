## Requirements

PHP v.8.1, composer

## Install

1. Clone this repository with

```
git clone https://github.com/Al-kand/php-project-lvl4.git
```

2. Go to game folder in

```
cd php-project-lvl4
```

3. Make setup

```
make setup
```

4. Run

```
make start
```

5. The application is available at `http://0.0.0.0:8000`
